# GoRide

**GoRide** is a web-based application designed to simplify cab booking services by offering a seamless user experience. This project is built using modern web technologies to ensure performance, scalability, and maintainability.

## Features
- Real-time cab booking with optimized routing
- User-friendly interface for easy navigation
- Live status updates on booking and ride progress
- Driver and passenger management for improved customer experience
- Secure payment integration

## Technologies Used

|----------------|--------------------------------------------------|------------|

| **HTML**       | Structuring the web pages                         | 31.9%      |
| **JavaScript** | Dynamic behavior and event handling               | 29.3%      |
